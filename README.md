# week5-library-system# Library Management System

## Project Description
This project is a Library Management System developed using Python and Object-Oriented Programming (OOP). The system allows users to manage books in a library by adding, viewing, and removing books.

## Features
- Add new books
- View books
- Remove books
- Save data using JSON file
- Menu driven interface

## Technologies Used
- Python
- JSON
- OOP Concepts

## How to Run

cd week5-library-system  
python library_system/main.py
